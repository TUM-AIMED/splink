sPLINK

sPLINK (safe PLINK) is a federated, privacy-preserving tool set based on the HyFed framework for genome-wide association studies (GWAS).

Unlike PLINK, sPLINK preserves the data privacy. In contrast to meta-analysis approaches which lose accuracy if the data distribution is imbalanced/heterogeneous across the cohorts, sPLINK is robust against data heterogeneity. That is, sPLINK gives the same results as aggregated analysis conducted with PLINK even if phenotype or confounding factor values have heterogeneously been distributed across the cohorts.
